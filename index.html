POC HTML Dialog

1. Modal — showModal()

Backdrop natif, trap focus, fermeture via Escape — tout géré par le navigateur.

Modal native

Naviguez avec Tab — le focus reste dans la dialog. Appuyez sur Escape pour fermer.

2. Non-modal — show()

Pas de backdrop, pas de blocage — la page reste interactive derrière.

Non-modale

Interagissez avec la page pendant que cette dialog est ouverte.

3. <form method="dialog">

La soumission ferme la dialog nativement et expose dialog.returnValue.

Formulaire natif

4. autofocus

Par défaut le focus va sur le premier élément focusable. autofocus permet de cibler un élément précis.

Focus par défaut

Le focus arrive sur le premier élément focusable (l'input).

Avec autofocus

Le focus arrive sur le bouton grâce à autofocus, pas sur l'input.
